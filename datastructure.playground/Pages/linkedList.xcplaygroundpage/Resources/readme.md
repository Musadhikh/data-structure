# Performance analysis

## Insertion
                    push        append      insert(after:)      node(at:)

Time Complexity:    O(1)        O(1)        O(1)                O(i), where is the given index



## Deletion
                    pop        removeLast      remove(after:)

Time Complexity:    O(1)        O(n)            O(1)
                                n=number of
                                nodes
